Pair War

Operating Systems project made in C.

Project Description

This project was built to understand how pthreads work within C. This project was designed in 2 iterations. The first iteration was developed to operate without pthreads in order to test the gameplay. The second iteration was made with 2 threads one for the card dealer and one that controls all of the players.
